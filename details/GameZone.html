Game Zone

Project Description:

This was the fourth Milestone project that I worked on during my Diploma in Full Stack Software Development with the Code Institute. It was part of the "Python and Data Centric Development" module.

The website is a book review and recommendation site.

The project was created with HTML, CSS, the Bootstrap CSS Framework, JavaScript, jQuery, Python and Django. SQLite was the default database used in development. PostgreSQL was the production database used through Heroku. Git and Github were used for version control.

Heroku was the cloud platform used to deploy and run the code pushed to the associated GitHub repository.

To find out more details about the project, you can view the Readme.md file by clicking the Github link below.
