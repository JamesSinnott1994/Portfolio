The Saw Doctors

Project Description:

This was the first Milestone project that I worked on during my Diploma in Full Stack Software Development with the Code Institute. It was part of the "User-Centric Front-end Development" module.

It is a website created for fans of “The Saw Doctors”, an Irish band from County Galway, who hold the record for Ireland’s all-time biggest-selling single “I Useta lover”.

The project was created with HTML, CSS, and the Bootstrap CSS Framework for developing a responsive and mobile-first site. Git and Github were used for version control.

To find out more details about the project, you can view the Readme.md file by clicking the Github link below.
