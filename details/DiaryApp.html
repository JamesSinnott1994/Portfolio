Diary App

Project Description:

Diary App created with Python using the Peewee ORM (Object Relational Mapping).

Project was created to demonstrate working with databases in Python.

User can add a new entry to their diary, view their entries, search for an entry, or delete an entry. All entries are stored in a database.

Project Duration:

September 2018 (2 days)
