Quiz Game

Project Description:

This was the second Milestone project that I worked on during my Diploma in Full Stack Software Development with the Code Institute. It was part of the "Interactive Front-end Development" module.

It was a fun interactive quiz game made using the Open Trivia Database REST API.

The project was created with HTML, CSS, JavaScript, jQuery and the Bootstrap CSS Framework. Git and Github were used for version control.

To find out more details about the project, you can view the Readme.md file by clicking the Github link below.
