Rainbow Run

Project Description:

This was a group project that I worked on in my fourth year in college.

Rainbow Run is a simple single screen 2D platformer. The game uses a simple 2D based art style with animated sprites. The main objective of the game is for the player to navigate through the obstacles in each level and to attempt to reach the exit marker. The Player must collect blobs of paint and will be constantly chased by darkness, which will slowly turn all colours in the game to black. If the player fades to black completely they will get a game over screen.

Project Duration:

November 2015 (2 weeks)

Technologies used:

  • C++
  • SDL
  • Github
  • Jira
