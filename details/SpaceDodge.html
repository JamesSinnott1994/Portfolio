Space Dodge

Project Description:

This was my main project for my final year in IT Carlow. Space Dodge is a 2D side-scrolling sci-fi platform shooter game where the player-controlled character must navigate various obstacles and overcome multiple enemies in order to progress through various levels.

When designing this game I went through the SCRUM agile software development process where I created a Game Design and Technical Design Document whilst also updating my tasks on JIRA.

The game was developed in Visual Studio 2013 with the C++ programming language, the Simple DirectMedia Layer (SDL) Framework and Box2D. Python and MySQL were used for the online leaderboard.

Project Duration:

September 2015 - May 2016

Technologies used:

  • C++
  • SDL
  • Python
  • MySQL
  • Github
  • Jira
