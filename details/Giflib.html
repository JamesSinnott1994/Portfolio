Spring GIF Library Project

Project Description:

My first web application created with Spring. It displays a number of GIFs on a homepage. You can click on an individual GIF for more details on it, search for a GIF, see the different categories of GIF, and view all the favourite GIFs.

Project Duration:

May - June 2020
